Notable Projects

AI-enabled optimized Test Selection for Codebase Changes

  • Developed a solution to identify an optimized set of tests to run in response to changes in the codebase, enhancing testing efficiency.
  • Integrated an AI classifier model that analyzes code coverage metrics and data from test cases, along with a generative AI model that leverages test and patch information from code commits.
  • Achieved targeted validation through the selection of an optimized test set, resulting in a ~72% reduction in machine time required for test execution without compromising quality.
  • Provided consumers with a dashboard in Splunk displaying AI prediction data, selected test sets, execution times, and associated savings, facilitating informed decision-making.
  • Implemented the solution within the code development team‚Äôs unit testing process and supported the validation team in their regression and functional validation efforts.

AI-Powered Automated Triage of Test Results

  • Contributed to the development of an AI solution for the automated triage of test results, enhancing efficiency in identifying and managing test failures.
  • Implemented a classifier model trained on potential failure patterns to categorize test execution results as True/False failures.
  • For all identified True failures, compared failure essences with existing bug clusters to identify duplicates; if none are found, a new bug is created for further investigation.
  • Utilized a generative AI model to create human-readable bug descriptions and details from test procedures and failure logs, streamlining the reporting process.
  • Achieved nearly 50% savings in human time spent on triaging test failures and reduced the incidence of false failures.
  • Provided consumers with visualizations of classifier predictions, clustering details, and time savings through a Splunk dashboard for enhanced decision-making.

NDNViber: Vibration Assited Automated Bootstrapping of IoT Devices

  • Developed NDNViber, a novel solution to enhance secure bootstrapping processes for resource-constrained and interfaceless IoT devices.
  • Addressed challenges associated with traditional bootstrapping methods by implementing a dynamic and secure out-of-band communication scheme using modulated vibrations, enabling multiple devices to bootstrap simultaneously.
  • Created a prototype utilizing a commodity smartphone as the controller to bootstrap various small IoT devices, followed by comprehensive performance analysis to validate effectiveness.

NDN-ABS: Attribute Based Signature scheme for Named Data Networking

  • Developed NDN-ABS, an innovative signature design for Named Data Networking (NDN) that addresses limitations of traditional RSA and ECDSA signatures, particularly in disruptive and ad hoc environments.
  • Eliminated the need for active communication channels to retrieve certificates, allowing for data packet verification without network retrieval, given a pre-configured trust anchor.
  • Enhanced producer anonymity by designing attributes that only identify application-defined high-level anonymity sets, ensuring individual producer identities remain confidential.
  • Implemented and visualized the prototype within a smart-campus environment, evaluating the NDN trust schema‚Äôs ability to manage the validity of NDN-ABS signatures.

Rapid Transient Trust Bootstrapping for NDN-Based Vehicular Networks

  • Investigated the establishment of secure communication in vehicular networks, focusing on trust management challenges due to the infrequent interactions and short-term relationships between vehicles.
  • Developed concepts based on the Swift Trust model to facilitate trust decisions in NDN-based communication, enabling vehicles within communication range to quickly assess trust for secure data publishing, consumption, and processing.
  • Employed a task-oriented approach to establish trust through request-response communication, enhancing reliability in dynamic vehicular environments.
  • Analyzed potential threats and attacks within this framework and discussed strategies for mitigation, contributing to the development of robust security measures in vehicular networks.

NachOS - Operating System Implementation

  • Modified the thread subsystem to support the creation and synchronization of multiple threads using semaphores, locks, and condition synchronization primitives.
